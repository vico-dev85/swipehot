# Research: Similar Projects & Visual Design Inspiration

## Context

I'm building a TikTok-style live cam roulette web app (mobile-first). Users see one random live performer at a time, full-screen, and swipe through them. All UI floats transparently over the video. The goal is a premium, modern, addictive experience — not a cheap-looking adult site.

I need to find existing apps, sites, and open-source projects that have similar interaction patterns so I can study their visual design, learn from their mistakes, and identify design elements worth borrowing.

I'm looking for visual inspiration across THREE categories:
1. Adult/cam sites with modern design
2. Non-adult apps with the same interaction pattern (full-screen vertical swipe)
3. Open-source implementations I can study
