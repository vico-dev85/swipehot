# Lovable Design Prompts for xcam.vip

## How to Use

1. Start a new Lovable project
2. Paste **Prompt 1** (roulette app) as the initial prompt
3. After Lovable generates it, iterate if needed ("make the CTA bigger", "darken the gradient", etc.)
4. Then paste **Prompt 2** (content pages) to add model page, blog post, and blog listing pages
5. Export the code and bring it back to this repo

## What Lovable Will Build (Design Only)
- Roulette overlay UI with placeholder video
- Model profile page
- Blog post page
- Blog listing/category hub page
- Shared design system (colors, fonts, components)

## What Lovable Will NOT Build
- Real API calls or backend logic
- Real video streaming / iframe embeds
- Personalization or A/B testing
- Analytics tracking
- Authentication

All data is placeholder. The design is what we're after.
