# Research Process

## Overview
Research prompts designed to inform the visual/UX design of xcam.vip before building.
Results feed into a Lovable AI design spec, then into the actual codebase.

## Workflow
1. Claude Code creates research prompts in `todo/`
2. Victor runs prompts through ChatGPT/Claude web
3. Results are pasted into `results/` (same filename)
4. Claude Code reviews results and writes the Lovable design spec

Run them in parallel if possible. Aim for 30-45 min each.
